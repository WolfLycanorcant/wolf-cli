## Configuration

### Tool Registry

The `search_web` tool is automatically registered with:
- **Risk Level**: SAFE (no confirmation prompts)
- **Category**: Web & Information
- **Default Results**: 10 (configurable via max_results parameter)
- **Max Results**: 50 (hard limit to prevent excessive queries)

### System Prompt (Web Mode)

When using `wolfw`, a specialized system prompt is temporarily applied:

> "You are in web-search mode. Use the search_web tool to retrieve up to 10 DuckDuckGo results for the user's query. Then synthesize a clear, helpful summary with the most relevant findings. Include key takeaways as bullet points and provide source URLs. If no results are found, explain and suggest alternative queries. Be concise and informative."

This prompt ensures the LLM:
1. Calls the search_web tool
2. Synthesizes results (not raw JSON)
3. Includes source URLs
4. Provides helpful fallback suggestions

## Known Limitations

1. **Internet Connectivity Required**: Web searches require active internet connection
2. **DuckDuckGo Rate Limiting**: High-volume queries may be rate-limited by DuckDuckGo
3. **Query Language**: Search quality depends on query clarity (LLM helps formulate good queries)
4. **Result Availability**: Some niche queries may return no results
