# Wolf CLI v0.3.1 - Windows Only Release

## Release Date
2025-11-10

## Summary

Version 0.3.1 focuses on enhancing the Windows experience, providing a robust command-line AI assistant with tool-calling and vision capabilities. This version ensures proper configuration management and a streamlined installation process for Windows users.

---

## What's New

### ‚úÖ Enhanced Configuration Paths (Windows)

**Problem Solved**: Previous versions might have had less robust configuration path handling.

**Solution**: Implemented a more robust config directory helper for Windows:

- **Windows**: `%APPDATA%\wolf-cli\config.json`
  - Example: `C:\Users\Administrator\wolf\AI\WORKBENCH\wolf-cli\config.json`



---

## Technical Changes

### Files Modified

1. **`wolf/utils/paths.py`**
   - Added `get_config_dir(app_name)` function
   - Uses `%APPDATA%` on Windows for configuration paths

2. **`wolf/config_manager.py`**
   - Removed hardcoded path: ~~`C:/Users/Administrator/wolf/AI/WORKBENCH/.wolf-cli-config.json`~~
   - Now uses: `get_config_dir("wolf-cli") / "config.json"`
   - Config directories created automatically on first run

3. **`wolf/providers/ps_client.py`**
   - Fixed hardcoded `C:\` in disk usage detection
   - Now uses `get_root_disk_path()` which returns `/` on Linux/macOS

4. **`setup.py`**
   - Updated OS classifiers to reflect Windows-only support

5. **`README.md`**
   - Updated platform support status (Windows only)
   - Documented configuration file locations for Windows

6. **`CHANGELOG.md`**
   - Added v0.3.1 release notes

### Windows


## Configuration

### Config File Locations

Configuration is stored in:

| Platform | Location | Environment Variable |
|----------|----------|---------------------|
| **Windows** | `%APPDATA%\wolf-cli\config.json` | `APPDATA` |

### Verify Config Path

```bash
# Check where config will be created
python -c "from wolf.utils.paths import get_config_dir; print(get_config_dir('wolf-cli'))"

# On Windows, should output: C:\Users\YourName\AppData\Roaming\wolf-cli
```

## Migration Guide

### For Existing Users (Windows)

If you're upgrading from v0.3.0 to v0.3.1 on Windows:

**Old config location**: `C:\Users\Administrator\wolf\AI\WORKBENCH\.wolf-cli-config.json`  
**New config location**: `%APPDATA%\wolf-cli\config.json`

Wolf will create a new config file with defaults. To migrate your settings:

1. Copy your old config file
2. Paste it to the new location: `%APPDATA%\wolf-cli\config.json`
3. Or reconfigure Wolf using environment variables
